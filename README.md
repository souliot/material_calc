# 环境准备

## 1、创建虚拟环境

```shell
conda create -n material_calc python=3.10
```

## 2、安装 FastApi、Uvicorn、python-multipart

```shell
# FastApi
pip install fastapi

# Uvicorn
pip install uvicorn

# python-multipart
pip install python-multipart
```

## 3、安装 pymatgen

```shell
# pymatgen
pip install pymatgen
```
